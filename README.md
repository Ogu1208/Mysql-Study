<details>
  <summary> 서버접속 </summary>
  
```
mysql -uroot -p
```
  `uroot` : root라는 사용자(user)로 접속하겠다. 
  **root**는 관리자로 모든 권한을 가지고있다.
  `-p` : 비밀번호입력(-p1234와 같이 바로 입력도 가능, but 보안성때문에X)
  `내 비밀번호 : 동생생일`
</details>
